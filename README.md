# listdiff

compute difference of two lists similar to [Elixir's `--` operator](https://elixir-recipes.github.io/lists/list-difference/)

## install
```bash
$ npm install listdiff
```

## usage
```javascript
const diff = require('listdiff')

console.log(diff([1, 2, 3], [3])) // [1, 2]
```
